utils package

This package provides various utility functions to support the PlanningTool backend. It includes modules for database operations and JSON handling.

Submodules

Below is a list of modules contained in this package.

database_helper

Configuration and path utilities for SimNIBS-based workflows.

This module reads configuration settings from config.ini and sets up directory paths (e.g., data path, code path, database directory). It also provides helper functions to retrieve simulation output paths and mesh file paths.
